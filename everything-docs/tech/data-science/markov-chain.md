# 马尔可夫链 (Markov Chain)：活在当下的数学模型

> **核心定义**：马尔可夫链是一个数学系统，它经历从一个状态到另一个状态的转换。其最著名的特性是**“无记忆性” (Memorylessness)**，即**“未来只基于现在来预测，而与过去无关”。**

## 1. 核心公式与通俗解释

*   **公式**：$P(X_{n+1} = x | X_1, X_2, ..., X_n) = P(X_{n+1} = x | X_n)$
    *   翻译成人类语言：明天($X_{n+1}$)发生什么的概率，只取决于今天($X_n$)是什么样，而不取决于昨天、前天或大前天发生了什么。

### 直观例子：醉汉走路 (Random Walk)
想象一个醉汉在街上走。
*   **状态**：他在路灯 A 的位置。
*   **预测**：他下一步是往左倒还是往右倒？
*   **马尔可夫性质**：这只取决于他**现在的姿势和重心**（State $t$）。至于他是从家里一路喝过来的，还是从酒吧刚出来的（历史路径），对他“下一步怎么倒”没有直接影响。

## 2. 为什么说《周易》是马尔可夫链？

大多数算命系统（如紫微斗数、八字）是**基于初始参数的宿命论**。
*   它们问：你出生年月是什么？（Input = History/Fixed Origin）
*   推导：因为你出生在那个时刻，所以你 30 岁会发财。

但《周易》不同，它是**基于当下的状态转移**。
*   **起卦**：你现在起到了“乾卦九五”。
*   **状态 ($State_t$)**：这是你**此刻**的能量切片（Position）。
*   **转移 ($Transition$)**：因为是“九五”（老阳），所以有一股向下的势能。
*   **未来 ($State_{t+1}$)**：这一爻变成了阴，卦象变为“大有”。

**《周易》不问出身**。它不在乎你是王子还是乞丐（过去），它只看你**这一刻**掷出的硬币是正面还是反面（当下）。它相信**当下蕴含了通往未来的全部信息**。

## 3. 常见应用
1.  **天气预测**：如果不考虑复杂的卫星云图，只看“今天下雨，明天有 70% 概率继续下雨”，这就是简单的马尔可夫模型。
2.  **输入法联想**：当你输入“我”，输入法推荐“是”或“想”。它不关心你十分钟前聊了什么，只关心你上一个字打了什么。
3.  **Google PageRank**：基于网页之间的链接跳转概率，本质上也是一种马尔可夫过程。
