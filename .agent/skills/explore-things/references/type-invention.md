# [中文标题] ([English Title]) - 万物拆解与溯源

> [!NOTE]
> [一句话简介/核心洞察：它是什么？它解决了什么本质问题？]

## 1. 视觉解构 (Visual Deconstruction)
*逻辑：想象一张“3D 爆炸图”或“生物解剖图”。*

*   **Skeleton (骨骼)**: 支撑结构是什么？（如自行车的车架，摩天大楼的钢筋）。
*   **Heart (动力心脏)**: 能量来源和转化中心在哪里？（如发动机，发条，芯片的时钟）。
*   **Limbs (执行肢体)**: 谁直接产生作用？（如车轮，电梯轿厢，显示像素）。
*   **Senses (感知/控制)**: 谁在指挥？（如恒温器，调速器，算法）。

## 2. 核心逻辑 (The Core Logic)
*逻辑：回到发明之前的世界，寻找“痛点”。*

*   **The "Ouch" Moment**: 人类面临的根本困境是什么？
*   **The Mission**: 这个发明的**本质目标**是什么？（例如：“效率放大器”或“能力延伸器”）。
*   **Problem Solved**: 它解决了哪 1-2 个最核心的物理难题？（如：火车解决了摩擦力，自行车解决了步行效率）。

## 3. 物理原理 (Physical Principles)
*逻辑：剔除装饰，直击本质。*

*   **Core Laws**: 列出它赖以生存的 2-3 个核心物理或数学定律。
*   **Metaphor Only**: **禁止**堆砌公式。必须使用“生活隐喻”来解释原理（如：陀螺效应像旋转的硬币）。

## 4. 连锁反应 (Action Flow)
*逻辑：慢动作回放用户操作后的瞬间。*

*   **The Spark**: 当用户按下“启动键”后的 0.1 秒内发生了什么？
*   **The Relay**: Step-by-step 描述能量/信息的接力过程。

## 5. 历史溯源 (Historical Origins)
*逻辑：从“史前”到“奇点”再到“进化”。*

*   **[史前] (Pre-existence)**: 在它出现之前，世界是什么样？人们怎么凑合过的？
*   **[奇点] (The Eureka Moment)**: 是谁？在什么情境下？突发奇想解决了问题（或关键突变）。
*   **[进化] (Milestones)**: 从雏形到现代形态，经历了哪 1-3 个关键的“基因突变”？（如：木轮 -> 充气轮胎）。

## 6. 蝴蝶效应 (The Butterfly Effect)
*逻辑：超越工具本身，看向文明深处。*

*   **Beyond Utility**: 不要只谈它的用途，要谈它如何改变了人类社会、文化或生物学。
*   **Examples**: （例如：空调改变了人口迁徙，香料推动了大航海，镜子催生了“自我意识”）。
*   **Impact**: 找出至少 3 个意想不到的深远影响。
