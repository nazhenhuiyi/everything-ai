# [中文姓名/名称] ([English Name]) - 人物解构与探索

> [!NOTE]
> [一句话定性：他是谁？他的核心身份标签是什么？（例如：“改变物理学图景的颠覆者”或“维多利亚时代的理性化身”）]

## 1. 身份解构 (Identity Deconstruction)
*逻辑：像建立角色卡一样拆解人物的基础属性。*

*   **Role (核心职能)**: 科学家 / 侦探 / 革命家 / 虚拟偶像...
*   **Era/Setting (时代/背景)**: 他身处什么环境？（例如：动荡的战国，充满迷雾的伦敦，早期的硅谷）。
*   **Archetype (原型)**: 在故事/历史中扮演了什么原型角色？（例如：智者，叛逆者，守护者）。

## 2. 核心驱动力 (The Core Drive)
*逻辑：挖掘人物内心深处的发动机。为什么他会成为他？*

*   **The "Why"**: 究竟是什么在驱动他行动？（是对真理的渴望？对秩序的执念？还是复仇？）。
*   **Philosophy (哲学观)**: 他的核心信条是什么？（例如：乔布斯的“极简主义”，福尔摩斯的“排除法”）。
*   **Unique Worldview (独特视角)**: 他眼中的世界与常人有何不同？

## 3. 关键作为 (Key Works & Acts)
*逻辑：通过行动定义人物，而非空洞的形容词。*

*   **Masterpiece (代表作)**: 具体的发明、理论、艺术作品或关键决策。
*   **Methodology (方法论)**: 他是如何做到的？有什么独特的“招式”？（例如：第一性原理）。
*   **Highlights**: 列举 1-3 个高光时刻 (High-light moments)。

## 4. 人生弧光 (The Narrative Arc)
*逻辑：人物的动态成长与变化轨迹。*

*   **Origin (起源)**: 这一切是从哪里开始的？（童年的创伤？偶然的启示？）。
*   **Struggle (困境)**: 他遭遇了什么巨大的阻碍或对手？
*   **Turning Point (转折点)**: 哪个时刻彻底改变了他的命运？

## 5. 蝴蝶效应与遗产 (Impact & Legacy)
*逻辑：人物消逝后，留给世界的痕迹。*

*   **Ripple Effect**: 他如何改变了行业、社会甚至人类认知？
*   **Cultural Symbol**: 他在今天代表了什么符号？（例如：爱因斯坦=天才）。
*   **Quote**: 引用一句最能代表他精神的名言。

## 6. 人性纹理 (Human Texture)
*逻辑：让神像落地，变回有血有肉的人。*

*   **Flaws (瑕疵)**: 他有什么性格弱点或怪癖？（例如：洁癖，傲慢，特定的饮食习惯）。
*   **Anecdote (轶事)**: 一个鲜为人知但极具画面感的小故事。
