# 复杂系统思维模型 (Complex System Model)

## 核心策略：系统动力学 (System Dynamics)
**原则**：关注变量之间的关系、反馈循环和演化趋势，而非单一因果。

## 适用场景
*   多变量交织的问题。
*   没有单一因果链的社会/经济/生态现象。
*   需要推演未来走向的问题。

## 调用思维层级
主要调用 **计算层** 和 **动力层**。

### 1. 计算层 —— 概率与演化 (Calculation Layer)
*   **非线性思维**：寻找系统中的“增强回路”（正反馈）和“调节回路”（负反馈）。
*   **临界点**：识别系统是否接近相变的临界点。
*   **长尾风险**：不仅关注平均值，更关注黑天鹅事件（幂律分布）。

### 2. 动力层 —— 激励与均衡 (Motivation Layer)
*   **纳什均衡**：系统中的各个体是如何博弈并达到暂时稳定的？
*   **激励机制**：是谁在推动系统运转？他们的KPI或利益是什么？
*   **古德哈特定律**：目前的评价指标是否导致了动作变形？

## 输出重点
*   绘制“变量关系图”（用文字描述）。
*   指出系统中的“杠杆点”（只需做微小改变就能引发布局变化的节点）。
