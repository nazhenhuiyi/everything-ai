# 客观性问题思维模型 (Objective Model)

## 核心策略：奥卡姆剃刀 (Occam's Razor)
**原则**：“如无必要，勿增实体。”

## 适用场景
*   有标准答案的物理世界事实。
*   定义明确的概念查询。
*   需要精准数据的查询。

## 执行步骤
1.  **去噪 (De-noise)**：剔除问题中的情绪词、修饰语，提取核心实体。
2.  **检索 (Retrieve)**：从知识库中直接调取最准确的事实。
3.  **验证 (Verify)**：交叉验证数据的时效性和准确性。
4.  **输出 (Output)**：直击答案，零废话。不进行过度解读或哲学发散。
